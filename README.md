## Tecnologías
- HTML
- CSS
- JavaScript

## Descripción
Plataforma para visualizar y gestionar eventos juveniles.

- main: rama principal
- inicio: página principal
- eventos: visualización de eventos
- perfil: gestión del usuario
- crear-evento: publicación de eventos
